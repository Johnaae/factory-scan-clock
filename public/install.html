Install Factory Scan Clock

Use these shortcuts so users can launch directly without typing the URL.

Install Manager App on Desktop (Chrome)

  1. Open /manager-login in Chrome.
  2. Click the install icon in the address bar (or menu > Cast, save, and share > Install page as app).
  3. Name it Factory Scan Clock - Manager and click Install.
  4. Pin the installed app to taskbar/start menu for one-click launch.

Install Kiosk App on iPad (Safari)

  1. Open /kiosk in Safari.
  2. Tap Share, then tap Add to Home Screen.
  3. Name it Factory Scan Kiosk and tap Add.
  4. Launch from Home Screen; it opens in standalone mode with minimal browser UI.
